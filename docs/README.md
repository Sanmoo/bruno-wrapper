# Demo

To record an interactive demo:

```bash
asciinema rec docs/demo.cast
# Run: bruwrapper run
# Select a collection, then a request
# Exit with Ctrl+D

# Convert to GIF:
agg docs/demo.cast docs/demo.gif
```

Alternatively, take a screenshot of the interactive mode and save as `docs/demo.png`, then update `README.md` to reference the PNG instead of the GIF.
